# Kieran计划展示页面

## 项目概述

Kieran计划展示页面是一个本地化全栈Web应用，允许用户上传文本和图片来填充创业计划内容，数据持久化保存，并支持动态交互式展示。

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **后端**: Python + Flask
- **数据存储**: 本地文件系统 (无数据库依赖)
- **架构**: 前后端分离设计，RESTful API交互

## 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd kieran-plan-display
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

## 运行方法

1. **启动Flask应用**
   ```bash
   python app.py
   ```

2. **访问应用**
   打开浏览器，访问 http://127.0.0.1:5000

## 功能说明

### 1. 编辑/上传模式
- **文本上传**: 支持多段文本输入（标题、简介、详细描述、里程碑等）
- **图片上传**: 支持创业相关图片，自动压缩到合适尺寸
- **实时预览**: 右侧画布实时显示已添加内容
- **拖拽功能**: 可自由拖动内容模块，位置自动保存

### 2. 展示/浏览模式
- **全屏画布**: 支持鼠标拖拽、缩放查看
- **模块详情**: 点击模块弹出详情卡片
- **连接线**: 按时间顺序连接相关模块，形成时间线

### 3. 核心功能
- **数据持久化**: 所有内容自动保存到本地`data/`目录
- **自动备份**: 每次保存时创建时间戳备份
- **图片处理**: 自动压缩图片到合适尺寸
- **错误处理**: 友好的前端错误提示
- **性能优化**: 图片懒加载、数据变更的防抖保存
- **安全性**: 文件上传类型验证、XSS防护

## 注意事项

1. **文件存储**: 所有上传的图片会保存在`uploads/`目录，文本内容会保存在`data/plan_data.json`文件中。

2. **备份机制**: 每次保存时，系统会自动在`data/backups/`目录创建时间戳备份，确保数据安全。

3. **图片限制**: 图片上传大小限制为5MB，支持的格式包括：png、jpg、jpeg、gif。

4. **安全性**: 系统已实现基本的XSS防护和文件类型验证，但在生产环境中建议进一步加强安全措施。

5. **性能优化**: 对于大量内容的场景，系统已实现图片懒加载和防抖保存，确保流畅的用户体验。

## 使用示例

1. **添加文本模块**:
   - 在编辑模式下，填写文本标题和内容
   - 点击"保存文本"按钮
   - 文本模块会出现在右侧画布上

2. **添加图片模块**:
   - 在编辑模式下，填写图片标题并选择图片
   - 点击"上传图片"按钮
   - 图片模块会出现在右侧画布上

3. **调整模块位置**:
   - 在画布上拖动模块到合适位置
   - 位置会自动保存

4. **查看模块详情**:
   - 点击任意模块
   - 会弹出模态框显示模块详细信息

5. **切换到浏览模式**:
   - 点击顶部导航栏的"切换到浏览模式"按钮
   - 进入全屏浏览模式，支持缩放和拖拽

6. **删除模块**:
   - 点击模块打开详情模态框
   - 点击"删除"按钮
   - 模块会被删除并从画布中移除

## 技术实现重点

1. **数据同步**: 前端位置变更后自动保存到后端
2. **图片处理**: 使用Pillow库进行图片压缩
3. **状态管理**: 前端使用简单状态模式管理展示/编辑模式
4. **拖拽功能**: 实现了基于原生JavaScript的拖拽API
5. **动画效果**: 使用CSS3 transition和transform实现平滑动画
6. **性能优化**: 图片懒加载、数据变更的防抖保存
7. **安全性**: XSS防护、文件类型验证
